### 🔬 [aitraining](https://github.com/iannwendy/aitraining)
> End-to-end research pipeline for detecting depression signs in Vietnamese YouTube comments

**Tech Stack:** Python · PyTorch · PhoBERT · BiLSTM · FastAPI · React

**Key Highlights:**
- Compared **4 model architectures**: TF-IDF+SVM, BiLSTM, PhoBERT, Domain-Adaptive PhoBERT
- Achieved **0.84 F1-macro** in-domain performance
- Discovered **0.50 generalization gap** between in-domain and cross-domain data
- Built comprehensive web demo with live predictions and BERTopic visualization
- Generated **125K+ labeled comments** dataset
